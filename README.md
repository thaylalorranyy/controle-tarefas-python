# Controle de Tarefas em Python

Projeto em Python para prática de lógica de programação. 

# Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

# Objetivo 
Aplicar conceitos de listas, funções e estruturas condicionais em Python no desenvolvimento de um sistema de controle de tarefas.
